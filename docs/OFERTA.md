## Proces zamówienia (wszystkie pakiety)

1. Wybór pakietu w `/uslugi` → koszyk  
2. Płatność Stripe  
3. E-mail: potwierdzenie + link do briefu (dane firmy, logo, treści, dostępy)  
4. Realizacja wg terminu pakietu  
5. Oddanie: URL produkcji + instrukcja + 30 dni poprawek w scope
