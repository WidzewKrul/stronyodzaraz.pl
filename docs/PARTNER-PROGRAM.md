# Partner program — cross-sell i affiliate

**Cel:** pasywne leady z księgowych, doradców BHP/HACCP, agencji komplementarnych.

---

## Model

| Partner | Prowizja | Cookie/attribution |
|---------|----------|-------------------|
| Księgowy / biuro rachunkowe | 10% pierwszej płatności | 90 dni |
| haccpnajuz / bhp / regulamin — internal | cross-link footer | brak prowizji |
| Freelancer dev (referral) | 5% | 60 dni |

**Wypłata:** po `DELIVERED` + 14 dni (brak chargeback). Min 200 zł wypłata.
